쿠팡진입점수 AI — 크롬 확장 설치 가이드
=========================================

1. Chrome 주소창에 입력: chrome://extensions
2. 우측 상단 "개발자 모드" 켜기 (토글 ON)
3. "압축 해제된 확장 프로그램 로드" 클릭
4. 이 폴더(extension/)를 선택
5. 확장 프로그램 아이콘이 주소창 오른쪽에 나타남

사용법
------
- 쿠팡 상품/검색 페이지에서 아이콘 클릭 → 자동 추출
- 또는 키워드를 직접 입력 후 "분석 시작"
- 무료: 하루 3회 분석
- 도매매칭·마진계산·등록전략 탭 순서로 활용
